# 项目实战-黑马智慧养老中心项目

## 课程介绍与项目展示

​	"智慧养老中心" 项目采用前沿技术栈 SpringBoot、MyBatisPlus（MyBatis） 作为后端基础，结合 Vue3、ElementPlus、TypeScript（JavaScript） 构建响应式前端界面。是目前最新技术栈，确保项目的稳定性与兼容性。数据库方面，我们选用 MySQL 作为关系型数据库，结合最新 ORM 框架，实现高效的数据访问和操作。

#### 1.1、后端核心技术栈

• SpringBoot：利用其轻量级特性快速构建RESTful API，简化部署与配置，提高开发效率。

• MyBatisPlus：基于MyBatis的增强工具，提供强大的CRUD操作，简化SQL编写，提升数据库交互效率。

• MySQL：作为关系型数据库存储用户信息、商品数据等，保证数据的持久化与高效查询。

#### 1.2、前端核心技术栈

• Vue3 + TypeScript/JavaScript：使用Vue3结合TypeScript，增强代码的可维护性和类型安全性。

• ElementPlus：提供丰富且美观的UI组件库，加速界面开发，确保良好的用户体验。

• uni-app：一套代码，多端运行，轻松构建 iOS、Android、H5、微信小程序等多个平台的应用，实现跨平台的无缝对接。

### 2、store下新建 collapse 目录，并建 index.js

```js
import { defineStore } from "pinia";
//定义store
export const collapseStore = defineStore('collapseStore',{
    state:()=> { //状态
        return{
            collapse:false
        }
    },
    actions:{ //方法
        setCollapse(collapse){
            this.collapse = collapse;
        }
    },
    getters:{ //属性
        getCollapse(state){
            return state.collapse
        }
    }
})
```

### 3、layout 下新建 Collapse.vue组件

```js
<template>
    <el-icon :size="30" style="cursor: pointer;color: #FFF;" @click="iconClick">
        <!-- 动态组件:有些场景会需要在两个组件间来回切换 -->
        <component :is="status ? Expand : Fold"></component>
    </el-icon>
</template>

<script setup lang="ts">
import { Expand, Fold } from "@element-plus/icons-vue";
import { computed } from 'vue';
import { collapseStore } from '@/store/collapse/index';
// 引入store
const store = collapseStore()
// 获取状态
const status = computed(() => {
    return store.getCollapse
})
// 点击图标
const iconClick = () => {
    store.setCollapse(!store.collapse)
}
</script>
```

```js
<template>
```

```js
      <el-breadcrumb-item v-for="item in tabs">{{ item.meta.title }}</el-breadcrumb-item>

  <script setup lang="ts">
  import { onMounted, ref, Ref, watch } from "vue";
  import { useRoute } from "vue-router";
  //获取当前路由
  const route = useRoute();
  //定义面包屑导航数据

  watch(
    () => route.path,
    () => getBredcrumb()
  );
  //获取面包屑导航数据

      //找出有title的数据

      console.log(first)
      if(first.path !== '/dashboard'){
          mached = [{path:'/dashboard',meta:{title:'首页'}} as any].concat(mached)
      }
      tabs.value = mached
  };
  onMounted(()=>{
      getBredcrumb()
  })
  </script>
```
